## 4. 影響度・緊急度の定義

### 影響度
| レベル | 定義 |
|--------|------|
| 高 | プロジェクト全体に影響、スケジュール遅延の可能性大 |
| 中 | 一部機能・チームに影響、代替手段で対応可能 |
| 低 | 軽微な影響、通常作業で対応可能 |

### 緊急度
| レベル | 定義 |
|--------|------|
| 高 | 即時対応が必要（24時間以内） |
| 中 | 今週中の対応が必要 |
| 低 | 計画的に対応可能 |
