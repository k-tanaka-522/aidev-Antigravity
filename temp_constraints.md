# 🔒 開発プロセス共通の制約事項 (Process Constraints)

## 📌 技術スタック
* **バックエンド:** Python 3.12 と Flask フレームワークを使用すること。
* **フロントエンド:** React と TypeScript を使用すること。
* **インフラ:** すべてのクラウドインフラは Terraform (IaC) で定義すること。

## 🛡️ セキュリティ基準
* すべてのAPIエンドポイントは、OAuth 2.0または同様の認証メカニズムを実装すること。
* **セキュリティエージェント**の明示的な承認なしに、新たな外部ライブラリを導入しないこと。

## 💡 コード品質要件
* **コーダーエージェント**が作成するすべてのコードは、最低 90% の単体テストカバレッジを達成すること。
* すべての新しい機能およびコンポーネントには、`README.md` または同等のドキュメントを含めること。

## 🔗 エージェントの連携ルール
* **PMエージェント**は、すべての設計フェーズが完了するまで、実装フェーズへの移行を許可しないこと。
* **テスターエージェント**は、E2Eテスト結果に重大な失敗がある場合、**QAエージェント**に通知し、デプロイメントを自動的にブロックすること。
