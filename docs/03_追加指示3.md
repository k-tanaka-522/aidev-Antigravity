# 追加指示：テンプレート配置場所の修正

## 概要

第二弾・第三弾で指定した `templates/` ディレクトリは、お客様納品時に不要なため、`.agent/` 配下に移動してください。

---

## 修正内容

**変更前**
```
{プロジェクトルート}/templates/
```

**変更後**
```
{プロジェクトルート}/.agent/templates/
```

---

## 理由

- `docs/` はお客様に納品する成果物のみを格納
- テンプレートはAI用の参照ファイルであり、納品物ではない
- `.agent/` 配下にまとめることで、圧縮して渡す際にクリーンな状態になる

---

## 最終ディレクトリ構成

```
{プロジェクトルート}/
├── GEMINI.md
├── .agent/
│   ├── workflows/        # ワークフロー定義
│   └── templates/        # テンプレート（AI参照用）
│       ├── 00_プロジェクト管理/
│       ├── 01_企画/
│       ├── 02_要件定義/
│       ├── 03_基本設計/
│       ├── 04_詳細設計/
│       ├── 05_製造/
│       ├── 06_試験/
│       ├── 07_リリース/
│       └── 08_保守運用/
└── docs/                 # 成果物（お客様納品対象）
    ├── 00_プロジェクト管理/
    ├── 01_企画/
    ...
```

---

**第二弾・第三弾の指示書内の `/templates/` パスは、すべて `/.agent/templates/` に読み替えてください。**
